#Тестовая БД для ДЭ	
База данных для хранения информации о товарах в  магазине бытовой  техники.
База данных имеет название "bitovaya_texnika"


## Таблицы базы данных

| name_tovar       | nvarchar(100)  | Название                      |

| Genre            | nvarchar(50)   |                   |
